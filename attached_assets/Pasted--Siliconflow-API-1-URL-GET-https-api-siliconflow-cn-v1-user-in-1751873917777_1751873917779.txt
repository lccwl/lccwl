硅基流动（Siliconflow）API 参考总结
 
一、用户与模型管理
 
1. 获取用户账户信息
- URL： GET https://api.siliconflow.cn/v1/user/info 
- 认证：需  Authorization: Bearer <token>  头部。
- 响应：返回用户基本信息（ID、名称、邮箱等）、余额（ balance 、 totalBalance ）及账户状态。
2. 获取模型列表
- URL： GET https://api.siliconflow.cn/v1/models 
- 参数：可通过  type （文本、图像等）或  sub_type （聊天、文本转图像等）过滤模型。
- 响应：返回模型ID（如  stabilityai/stable-diffusion-xl-base-1.0 ）、创建时间等信息。
 
二、批量任务（Batch）管理
 
1. 取消批量任务
- URL： POST https://api.siliconflow.cn/v1/batches/{batch_id}/cancel 
- 参数：需指定  batch_id 。
- 响应：返回任务状态（如  in_queue ）、输入/输出文件ID等。
2. 获取批量任务列表/详情
- 列表： GET https://api.siliconflow.cn/v1/batches ，返回任务列表及分页信息。
- 详情： GET https://api.siliconflow.cn/v1/batches/{batch_id} ，返回单个任务的完整信息（状态、时间戳、错误信息等）。
3. 创建批量任务
- URL： POST https://api.siliconflow.cn/v1/batches 
- 参数：需  input_file_id （输入文件ID）、 endpoint （如  /v1/chat/completions ）、 completion_window （处理时间窗口，如  24h ）。
4. 文件管理
- 上传文件： POST https://api.siliconflow.cn/v1/files ，支持  batch  用途的文件上传。
- 获取文件列表： GET https://api.siliconflow.cn/v1/files ，可通过  purpose=batch  过滤。
 
三、多媒体生成
 
1. 视频生成
- 提交生成请求： POST https://api.siliconflow.cn/v1/video/submit ，需指定模型（如  Lightricks/LTX-Video ）和prompt，返回  requestId 。
- 查询状态： POST https://api.siliconflow.cn/v1/video/status ，用  requestId  查询视频URL（有效期1小时）。
2. 音频处理
- 语音转文本： POST https://api.siliconflow.cn/v1/audio/transcriptions ，上传音频文件并指定模型（如  FunAudioLLM/SenseVoiceSmall ）。
- 文本转语音： POST https://api.siliconflow.cn/v1/audio/speech ，支持模型（如  fishaudio/fish-speech-1.5 ）、语音风格、采样率等参数。
- 参考音频管理：可上传、删除、查询预置音色（ uri  标识）。
3. 图像生成
- 创建图片生成请求： POST https://api.siliconflow.cn/v1/images/generations ，需模型（如  deepseek-ai/Janus-Pro-7B ）和prompt，返回图片URL（有效期1小时）。
 
四、文本处理
 
1. 重排序请求
- URL： POST https://api.siliconflow.cn/v1/rerank 
- 参数：需查询词（ query ）、文档列表（ documents ），返回相关度排序结果及分数。
2. 嵌入向量生成
- URL： POST https://api.siliconflow.cn/v1/embeddings 
- 参数：指定模型（如  BAAI/bge-large-zh-v1.5 ）和输入文本，返回向量表示及 token 消耗。
3. 聊天对话请求
- URL： POST https://api.siliconflow.cn/v1/chat/completions 
- 参数：支持多模型（如  Qwen/QwQ-32B ）、消息历史（ messages ）、流式输出（ stream ）等，返回对话响应及 token 统计。
 
五、AskMany AI 平台说明
 
- API 地址： https://open.askmany.chat/ 
- 账号体系：与 AskManyAI 网站账号不互通，需重新注册。
- 准备工作：生成 API 密钥（格式为  sk-xxxxxxxx ），需妥善保管以防盗刷。
 
核心特点总结
 
1. 多模态支持：覆盖文本、图像、音频、视频生成及处理，支持批量任务高效管理。
2. 认证统一：多数接口通过  Bearer Token  认证，需在请求头中携带 API 密钥。
3. 有效期提醒：视频、图片 URL 有效期为1小时，需及时转存；批量任务有处理时间窗口限制。
4. 模型动态调整：部分模型（如音频、视频模型）可能不定期更新，需关注官方公告。
